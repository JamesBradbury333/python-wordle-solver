# Python Wordle Solver
## Getting started
To get the project started you need pipenv installed and running
```shell
snap install pipenv
```
First sync the dependencies
```shell
pipenv sync --dev
```
Then you should be able to run the unit-tests succesfully
```shell
pipenv run unit-test
```
